# 🏗️ Desenvolvimento Low-Code (Low-Code Development)

## 💡 O que é

**Low-code** é uma abordagem de desenvolvimento de software que minimiza a quantidade de código escrito manualmente, substituindo-o por interfaces visuais, componentes prontos e configurações. O objetivo é acelerar a criação de aplicações e torná-la acessível a pessoas com pouca ou nenhuma experiência em programação tradicional.

A diferença entre low-code e no-code está no grau de abstração: plataformas **no-code** não exigem nenhum código; plataformas **low-code** permitem combinar componentes visuais com trechos de lógica quando necessário. O Kodular se posiciona entre os dois — é essencialmente no-code para quem usa apenas blocos, mas permite extensões com código Java para usuários avançados.

O movimento low-code ganhou força nos últimos anos como resposta à escassez de desenvolvedores e à necessidade de prototipação rápida em contextos acadêmicos, startups e transformação digital de pequenas empresas.

## ⚙️ Como é usado neste projeto

O app **Catálogo de Livros** foi desenvolvido inteiramente no **Kodular**, uma plataforma low-code baseada no MIT App Inventor. Nenhuma linha de código foi escrita manualmente — toda a interface e a lógica foram construídas arrastando, soltando e conectando componentes e blocos visuais.

Isso permitiu que o app com CRUD completo + integração Firebase fosse construído em um único período de aula, com extensões adicionadas de forma independente sem necessidade de conhecimento de Kotlin ou Java.
